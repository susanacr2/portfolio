How to make the site live Step by step
1

Create a GitHub account

Go to github.com and sign up for free. This is where your site files live.

2

Create a new repository

Click "New repository" → name it portfolio → set it to Public → click Create.

3

Upload all your files

Click "Add file → Upload files". Drag and drop all files from your site folder: index.html, admin.html, susana.avif, community1.png, community2.png, AURO.png, auro_onboarding.png, auroexpand.png

4

Connect Cloudflare Pages

Go to pages.cloudflare.com → Create a project → Connect to Git → Select your portfolio repo → leave all settings default → click Save and Deploy. You'll get a free URL like portfolio.pages.dev in about 2 minutes.

5

Optional: add your custom domain

Buy susanachinchilla.com on Cloudflare for ~$12/year. Then in Cloudflare Pages → Custom domains → add your domain. Done. No other costs ever.

✓

To update your site in the future

Go back to GitHub → click the file you want to change → click the pencil edit icon → make changes → click "Commit changes". Cloudflare automatically rebuilds and publishes in ~1 minute.
