Deployment steps
1

Create a GitHub account

Go to github.com and sign up for free.

2

Create a new public repository

Click "New repository" → name it portfolio → set to Public → click Create.

3

Upload all your files

Click "Add file → Upload files" → drag all files from your folder → click "Commit changes."

4

Connect Cloudflare Pages

Go to pages.cloudflare.com → Create a project → Connect to Git → select your repo → leave defaults → Deploy. You get a free URL like portfolio.pages.dev in ~2 minutes.

✓

To update in the future

Go to GitHub → open any file → click the pencil icon → edit → Commit. Cloudflare auto-rebuilds in ~1 minute.

5

Optional: custom domain

Buy susanachinchilla.com on Cloudflare for ~$12/year → Cloudflare Pages → Custom domains → add your domain. Done.
